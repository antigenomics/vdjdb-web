Contributing

Users interested in contributing TCR sequences of known antigen specificity to the database should read the instruction at the following link. To submit the contribution use the issue system of the VDJdb repository. If you are not familiar with GitHub you can file your submission by creating a new issue and attaching your files. Advanced users can use the pull-request system.

Software

Standalone application

A command line interface software (VDJdb-standalone) should be preferred in cases when huge amounts of immune repertoire sequencing data are being analyzed or when users are interested in obtaining unprocessed output for carrying out statistical analysis of the results on their own. The corresponding software package is available here. VDJdb-standalone is distributed as an executable JAR and requires Java v1.7+ to run, for additional details see the README file in the repository linked above.

Web server (Advanced)

VDJdb web browser application is an open-source software implemented using Play Framework, Angular and Semantic UI. A standalone browser version of the software can be installed locally, the following link contains the software package and installation instructions.

Other

You might be also interested in our lightweight immune repertoire browser VDJviz and our immune repertoire sequencing data analysis software VDJtools.

Public API and documentation

VDJdb web browser application uses public and fully documented API to query the database. The documentation, API description and usage examples can be found at ReadTheDocs website.
